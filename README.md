# GT-MP Basic Map Editor
A very simple Map Editor for GT-MP

**Min. Required GT-MP Version:** 0.1.7.x

## Commands
### General
`/createmap <fileName>`

`/loadmap <fileName>`

`/finishmap <fileName`

### Object Placer

`/addobject <objectHash>`

`/deleteobject <objectId>`

`/addobjectcategory <name>`

`/addobjecttocategory <categoryName> <modelHash> <objectName>`

`/savemodellist`

`/editobject <objectId>`

`/exiteditor`

## Controls
### General
`F2` Open Main Menu
### Object Placer
`NumPad0` Switch Rotation / Position Mode

`Numpad8` X-Axis+

`Numpad2` X-Axis-

`Numpad4` Y-Axis+

`Numpad6` Y-Axis-

`Numpad+` Z-Axis+

`Numpad-` Z-Axis-
